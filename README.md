# Steps to run the program

## Navigate to the degrees folder
```
$ cd degrees
```

## Compile the program
```
$ go build -o degrees ./main.go

```

## Run the executable by providing person urls as input
```
$ ./degrees amitabh-bachchan robert-de-niro
```
